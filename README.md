# Saloon Marketplace

Thiaza is a saloon marketplace where customers can book appointments and barbers can manage their schedules.

### Features:
- Authentication
- Role based authorization
- Google Maps location search
- Appointments Booking
- Store management
- Stripe payment module
- etc

### Languages:
##### Client Side:
- React.Js
- Redux
- Intl
##### Server Side:
- Express.Js
- Node.Js
- MongoDB

##### Third Party:
- Stripe 
- Cloudinary
- Google Maps
